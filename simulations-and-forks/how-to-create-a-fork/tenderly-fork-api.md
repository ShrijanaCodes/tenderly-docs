# Tenderly Fork API

The fork API has 5 endpoints:

**(1) Create Fork:** `POST /api/v1/account/{{accountID}}/project/{{projectID}}/fork`

\
Example request:

```
{
    "block_number": null,
    "network_id": "42",
	  "transaction_index": null,
		"initial_balance": 10000,
    "chain_config": {
        "chain_id": 3030,
        "homestead_block": 0,
        "dao_fork_support": true,
        "eip_150_block": 0,   
        "eip_150_hash": "0x41941023680923e0fe4d74a34bdac8141f2540e3ae90623718e47d66d1ca4a2d",
        "eip_155_block": 10,
        "eip_158_block": 10,
        "byzantium_block": 1700000,
        "constantinople_block": 4230000,
        "petersburg_block": 4939394,
        "istanbul_block": 6485846,
        "berlin_block": 9812189
    }
}
```

`block_number` , `initial_balance` (given in ETH, defaults to 100 if no parameter is provided) and `transaction_index` are optional parameters.\

**(3) Simulate Transaction:**

`POST /api/v1/account/{{accountID}}/project/{{projectID}}/fork/{{forkID}}/simulate`

\
Example Request:

```
{
    "block_number": null,
    "transaction_index": null,
    "from": "0x0000000000000000000000000000000000000000",
    "to": "0xBd770416a3345F91E4B34576cb804a576fa48EB1",
    "input": "0xef690cc0",
    "gas": 1000000,
    "gas_price": "0",
    "value": 0,
    "save": true,
    "root": "73af0de5-c634-481a-bbe0-445da87b329d"
}
```

**\[See Simulate API docs for a more detailed documentation]**\
